Пояснительная записка к ДЗ3 по АВС Татьянченко Дмитрий БПИ207

Здание выполнено на языке python 3.8.3

Как запустить программу:

	Используются стандартные флаги:
		-f <input file name> <first output file name> <second output file name> - считывание контейнера из текстового файла (пример - см. папку tests)
		-n <number on randomly generated instances> <first output file name> <second output file name> - запуск случайной генерации с заданным числом объектов

Как работает программа:
	
	сценарий аналогично предыдущим дз.
	Из особенностей можно выделить наследование от общего абстрактного класса без необходимости создавать отдельное поле для хранения типов в наследниках
	для последующей идентификации их в контейнера (удобно - позволяет динамическая типизация),
	итерируемый объект контейнера на базе стандартного списка (тоже удобно - делается в 2 строчки),
	ide не подсвечивает потенциальные баги, связанные со сравнением числа с объектом функции (неудобно - расплата за пункт 1). 

Время выполнения от кол-ва объектов:

	10 объектов ~ 0.0006 seconds
	100 объектов ~ 0.0017. - 0.0030 seconds
	1000 объектов ~ 0.0208 - 0.0098 seconds
	10000 объектов ~ 0.1144 - 0.0880 seconds
	100000 объектов ~ 0.8631 seconds
	1000000 объектов ~ 8.7965 seconds (ya skorost')

Вывод: пишется быстро и приятно, работает медленно и пофиг.
